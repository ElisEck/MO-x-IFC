Abstract

This ontology contains all models present in the Modelica LibEAS, which is an internal library at Fraunhofer IIS/EAS for the purpose of HVAC plant simulation. The ontology contains the models with their components and connections. The ontology was automatically created using the MoTTL-Transcriptor. Since it is large only one serialization (ttl-Syntax) is provided.
